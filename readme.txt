Plugin for adding tracking code to the header and/or footer.

== Description ==

<p>Plugin for adding tracking code to the header and/or footer which based on Khaled Hossain Saikat's great plugin Tracking Code 1.0.0.</p>


== Installation ==

1. Upload and extract `idienstler-tracking-code.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your tracking code by navigating Settings >> Tracking Code
